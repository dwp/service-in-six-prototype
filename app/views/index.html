Initial concept prototypes

These are links to the concept of applying for New Style benefits from the point that an account has been created.

Some of the functionality that was in the original individual protoypes has been stripped out to enable more rapid prototyping where there I encountered any code conflict.

So in effect, each landing page you will see from here would consitute the first page after you have successfully created an account and logged in.

At the point that this prototype was created, it was perceived that a user would have passed some pre-account eligibility questions, created an account and got to the stage where this begins.

In a real life scenario a user would be able to start anywhere in the journey and complete the to-dos in any order. But for speediness in this prototype they can only go in a linear order

As well as the end to end apply journey, which has extra questions compared to the current live journey, we also tested the uploading evidence pattern that comes up after the initial gather stage if appropriate.

Neil Holroyd – interaction designer
